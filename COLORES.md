# Paleta de colores

| Nombre           | Elementos                                          | HEX       |
| ---------------- | -------------------------------------------------- | --------- |
| Negro            | Fondo principal, color de texto en botones dorados | `#0D0D0D` |
| Oro              | Títulos, bordes, botones principales               | `#C6A667` |
| Verde            | Botones de éxito (`.btn-success`)                  | `#0F8F57` |
| Rojo             | Botones de peligro (`.btn-danger`)                 | `#B42222` |
| Plateado         | Texto secundario, fondos secundarios               | `#E4E4E4` |
| Oro Claro        | Hover de los botones dorados                       | `#E3C98C` |
| Negro (Absoluto) | Variante más oscura del negro élite                | `#000000` |
| Gris Oscuro      | Variante para textos o elementos desactivados      | `#8C8C8C` |
| Blanco           | Texto principal sobre fondos oscuros               | `#FFFFFF` |
| Gris (Card BG)   | Fondo para componentes tipo "card"                 | `#1A1A1A` |
| Gris (Bordes)    | Bordes suaves para tablas e inputs                 | `#CCCCCC` |
| Gris (Inputs BG) | Fondo para inputs de formularios                   | `#1b1b1b` |
